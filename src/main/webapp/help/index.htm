1       前言

本手册向用户描述如何使用信息系统运行维护平台。

1.1 预期读者

本手册主要针对《广州宝城综合信息管理系统》所有使用者的使用手册。

1.2 支持和帮助

除了本手册文档之外，系统使用过程中也随时可以获取帮助信息；如果遇到特殊问题，请联系系统管理员。

若需获得其他支持，可直接联系宝城公司客户服务部。

公司网址：http://www.bctaxi.com.cn/

联系电话：020-81800088

2       系统概述

《宝城综合信息管理系统》实现所有与公司主营业务相关的营运信息综合管理，将业务流程标准换、流程化处理，保证营运数据查询实时便捷、营运流程实施高效。

3       客户端要求

客户端计算机最低配置要求为：内存不少于512M，并有足够的硬盘空间。

客户端计算机必须安装以下软件：

3.1.1 浏览器

Chrome12+、Firefox10+、Safari5.1+、Opera11.50+、IE8+。

安装其中的任何一种浏览器均可，使用较老的版本可能会遇到不可预知的问题或者无法正常使用系统。有些高级功能在IE8中无法使用，如果要使用全部功能或得到更好用户体验和运行速度，推荐使用上述的非IE浏览器或使用IE9+。

如果你使用其它的双核浏览器，请务必切换到高速模式访问本系统。

3.1.2 操作系统

能安装上述浏览器的任何操作系统，如Windows XP、Win7、Ubuntu Desktop、Mac OS、openSUSE、SUSE Linux Enterprise Desktop、Redhat Linux Enterprise Desktop。

4       登录系统

在浏览器的地址栏中输入“http://192.168.0.253/bcsystem”，按下回车键即可打开系统的登录界面：

1)   在帐号栏中，请输入分配给你的登录帐号。

2)   在密码栏中，请输入你的帐号密码。

3)   点击【登录】按钮或按回车键，即可。

不清楚访问地址、帐号、密码等信息时，请联系系统管理员。
